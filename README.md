# Описание проекта "Habit GRabbit"

## Проблема

Разработчики могут часто терять фокус и выгорать при рутинных работах, все углубляет если они не видят свой прогресс - все это чревато прокрастинацией. Существующие инструменты как GitHub или другие приложения для "Вырабатывания привычки" - не вырабатывают привычки и не мотивируют на ежедневные разработки.

## Решение

Habit GRabbit это веб-приложение, которое через интеграцию с GitHub анализирует реальную активность разработчика (коммиты, пулл-реквесты, создание issues). На основе этих данных AI помощник (GPT или другая модель) дает персональные советы, хвалит за серии продуктивных дней (streak), помогает ставить цели и борется с прокрастинацией. Это как Duolingo, но для программистов.

**Целевая аудитория:**

* Студенты IT-специальностей.
* Junior-разработчики, которые хотят системно подойти к развитию.
* Разработчики, которым не хватает мотивации для личных проектов.

# User Stories

### 1) Как разработчик, я хочу ставить себе цели и получать напоминания, чтобы не забывать о делах и видеть их прогресс выполнения

### 2) Как разработчик, я хотел бы видеть свою ежедневную текущую серию дней без пропусков, чтобы поддерживать продуктивность и мотивацию

### 3) Как разработчик, я хочу авторизоваться через свой Github аккаунт, чтобы система автоматически получала данные о моей работе без ручного ввода

### 4) Как пользователь - я хочу общаться с AI ассистентом, который анализирует мою активность, чтобы получать персональные советы, поддержку и рекомендации

# Список сущностей и их связи

### User (Пользователь приложения)

Имеет один Github - аккаунт, много: записей, целей, сообщений с AI помощником

### GitHubProfile (данные, полученные при OAuth логине)

Имеет связь Один к одному с User.

### DailyStats (статистика за конкретный день)

Имеет связь Многие к одному с User

### Goal (цель или задачи - от User или AI помощника)

Состоит из: Дата, Число коммитов, issue, пулл-реквесты, выполнение о дневной норме.

Имеет связь Многие к одному с User.

### ChatMessage (сообщеня в диалоге с AI помощником)

Состоит из: роль (User / AI), текст, timestamp

Связь: Многие к одному с User.

# Ключевые экраны приложения

### 1) Авторизация (Login)

Кнопка войти через Github. Перенаправляет на Github OAuth, получает токен.

### 2) Дашборд (Dashboard)

Основной экран после входа. Тепловая карта активности (как календарь активностей в Github), текущий streak, круговой индикатор выполнения дневной нормы (норму - ставит сам пользователь при желании), краткий совет от AI на сегодня.

### 3) Чат с AI помощником

Интерфейс диалога. Пользователь может задать вопрос или просто запросить анализ. AI отвечает на основе статистики и целей. Кнопки быстрых действий, по типу: Проанализируй неделю

### 4) Список целей

Список (или таблица) текущих и завершенных целей. Возможность: создать новую цель, отметить выполнение и привязать ее к Github issue, переименовать текущие, удалить и создать новые колоны в таблице.

### 5) Профиль и настройки

Информация о Github аккаунте, настрйоки уведомлений, подключение / переподключении интеграции

### P.S. Объяснение названия "Habit GRabbit"

Habit - Привычка

**GRabbit** - игра слов (Git + Rabbit). Git - потому что работает на прямую с активностью пользователя GitHub, Rabbit - заяц ассоицируется с скоростью. (Иначе GRabbit - можно прочитать как "grab it" - возьми это)

**Получаем:** Привычка быстрого Git || Бери эту привычку :)))

Плюс: Два слова окончаются на "-it" - что является легкой рифмой
